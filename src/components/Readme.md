## 组件目录结构

```bash
|--- BaseFormComponent          ## 提供统一调用的基础组件
|--- BaseList                   ## 提供统一的 list 页面
|--- BaseModalForm              ## 提供统一的弹窗表单结构
|--- BaseQuery                  ## 提供统一查询表单结构
|--- BaseTable                  ## 提供统一的表单展示形式
|--- BusinessComponents         ## 存放业务组件
|--- ConfigComponents           ## 存放通过配置的生成的组件
|--- DataSelect                 ## 拉取后端接口展示选项的下拉框（后期将废弃）
|--- GeneralComponents          ## 提供通用组件，例如 checkbox_with_input
|--- GeneralComponents          ## 提供通用组件，例如 checkbox_with_input
|--- selects                    ## 历史遗留，后期慢慢优化
|--- .......                    ## 其它系统自带组件
```
